# Visages

FR
Appli de création et d'utilisation de flashcards pour apprendre rapidement les prénoms/noms d'élèves d'une classe.
Les données (noms et photos des élèves) sont stockées uniquement en local sur votre appareil.
Attention : il est important d'exporter les données de votre classe après création dans l'appli, pour les récupérer facilement si vous effacez l'historique ou le cache de votre navigateur.

Utilisable sans téléchargement au lien : https://rafsay203.github.io/visages/

Codé avec Claude AI.

EN
App for creating and using flashcards to quickly learn the names of students in a class.
The data (names and photos of students) is stored locally on your device only
Warning: make sure to export your class data after creating it in the app, so you can easily re-import it if your browser cache or history gets cleared.

Usable without download at the link: https://rafsay203.github.io/visages/

Coded with Claude AI.
